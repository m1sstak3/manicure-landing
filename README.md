# Nail Aesthetic - Premium Nail Salon Landing Page

![Hero Image](assets/images/hero.png)

## Описание проекта
**Nail Aesthetic** — это современный, отзывчивый и премиальный лендинг для студии ногтевого сервиса. Проект разработан с фокусом на демонстрацию качества услуг, безопасности процедур и удобства для клиента. 
Интерфейс спроектирован с учетом высоких стандартов UX/UI, чтобы стимулировать доверие и максимизировать конверсию в онлайн-запись (от 10%).

> ⚠️ **Примечание для проверяющих:** 
> Проект представляет собой Frontend-реализацию. Все данные клиентов (номера телефонов, адреса) и ссылки на мессенджеры в коде являются тестовыми заглушками (dummy-data), используемыми исключительно для демонстрации в портфолио.

## 🎯 Цели проекта
1. Предоставить пользователю понятную и структурированную информацию об услугах мастера маникюра и педикюра.
2. Повысить доверие к студии за счёт демонстрации стандартов безопасности (Стерелизация, СанПиН).
3. Обеспечить удобную онлайн-запись на любой платформе.

## 🛠 Стек технологий (Tech Stack)
- **HTML5:** Семантическая разметка (Semantic HTML), микроразметка Schema.org для локального бизнеса (LocalBusiness).
- **CSS3 (Vanilla):** CSS Variables (Custom Properties), Flexbox, CSS Grid Layout, адаптивная верстка (Mobile First & Media Queries).
- **JavaScript (ES6):** Vanilla JS без jQuery. Использование IntersectionObserver для scroll-анимаций.
- **Интеграции:**
  - `Flatpickr` — премиальный выбор даты для формы.
  - OpenGraph метатеги для красивых сниппетов в мессенджерах.

## 🏗 Архитектура и функционал
- **Модульная структура:** Четкое разделение ответственности (Separation of Concerns). Скрипты и стили вынесены в отдельные файлы без инлайн-примесей. Логика изолирована от глобальной области видимости `window`.
- **Responsive Design:** Лендинг адаптируется под смартфоны, планшеты и десктопы (от 320px до 1920px). Реализовано мобильное off-canvas меню ("бургер").
- **Scroll Animations:** Динамическое появление контента при скролле с использованием `IntersectionObserver API` для высокой производительности (Smooth UI/UX).
- **Галерея (Lightbox):** Кастомная реализация модальных окон для просмотра изображений портфолио без сторонних тяжеловесных библиотек.

## 🚀 Инструкция по запуску
Проект не требует сборки (No Build Tools Required). Для запуска:
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/ВашАккаунт/название-репозитория.git
   ```
2. Откройте файл `index.html` в любом современном браузере.
   *(Для работы локальных скриптов без ограничений CORS рекомендуется использовать расширение `Live Server` для VS Code).*

## 📈 Возможности масштабирования
1. **Мульти-филиальность:** Текущая структурная логика позволяет легко дублировать секции услуг под различные города или студии.
2. **Интеграция с backend:** Форма записи подготовлена под подключение реального REST API:
   - Внедрена симуляция отправки формы с асинхронными задержками (`setTimeout`).
   - Настроена базовая защита от повторных кликов при отправке (предотвращение брутфорс-нажатий).
3. **SEO Readiness:** Добавлены базовые метатеги, Description и Schema.org (LocalBusiness) с расчетом на кластеризацию под региональные запросы.
