## 📖 Описание проекта

Проект разработан с фокусом на демонстрацию качества услуг, безопасности процедур и удобства для клиента. Интерфейс спроектирован с учетом высоких стандартов UX/UI, чтобы стимулировать доверие и максимизировать конверсию в онлайн-запись.

> ⚠️ **Примечание:** Проект представляет собой Frontend-реализацию. Все данные клиентов и ссылки на мессенджеры являются тестовыми заглушками (dummy-data), используемыми исключительно для демонстрации в портфолио.

### 🎯 Цели проекта
1. Предоставить пользователю понятную и структурированную информацию об услугах.
2. Повысить доверие к студии за счёт демонстрации стандартов безопасности (Стерилизация, СанПиН).
3. Обеспечить интуитивно понятную онлайн-запись на любой платформе.

## 🛠 Стек технологий (Tech Stack)

* 🧱 **HTML5:** Семантическая разметка (Semantic HTML), микроразметка `Schema.org` для локального бизнеса (LocalBusiness).
* 🎨 **CSS3 (Vanilla):** CSS Variables (Custom Properties), Flexbox, CSS Grid Layout, адаптивная верстка (Mobile First & Media Queries).
* ⚡ **JavaScript (ES6):** Vanilla JS без jQuery. Использование `IntersectionObserver` для scroll-анимаций.
* 🔌 **Интеграции:**
  * `Flatpickr` — премиальный UI-компонент выбора даты для формы.
  * OpenGraph метатеги для красивых сниппетов в мессенджерах.
